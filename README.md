# 无限流Agent

## What is this project?
这是一个预测股票市场走向的投资决策平台。我希望它能在一定程度上有预测能力，帮助人们合理投资，帮助用户根据市场数据、风险偏好、新闻信息和历史表现做出更加合理的投资决策。
整个项目会随着开发不断升级，从最基础的规则决策开始，逐步加入强化学习、深度学习、LLM 和多 Agent 协作。

## Current Version

第一阶段不接入真实股票数据。
实现一个简化的投资环境，包括：

- 现金

- 股票

- 买入

- 卖出

- 持仓

- 收益计算

让 Agent 可以在这个环境中完成最基本的投资决策。

## Planned Features

- Rule-based Agent

- Q-learning Agent

- DQN Agent

- LLM Decision Agent

- Multi-Agent Investment System
